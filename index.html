What is this?

The Experiment

In 2005, Kyle MacDonald started with a single red paperclip and traded it ‚Äî one swap at a time ‚Äî all the way up to a house. AgentORP is the AI version of that experiment. It takes a starting item, sets a goal value, and autonomously works to get there through a series of trades on real marketplaces like Facebook Marketplace, Craigslist, OfferUp, and Mercari.

It's not a bot that just scrapes data. AgentORP thinks, plans, and acts. It searches for undervalued items. It estimates fair market value using sold listings across platforms. It writes listing descriptions, sends messages to real people, negotiates prices, and proposes trades. It knows when to push for a better deal and when to walk away.

How it works

Powered by OpenClaw

Under the hood, AgentORP runs on OpenClaw ‚Äî an open-source AI agent gateway that connects large language models to real tools like web browsers, messaging platforms, and file systems. OpenClaw gives the AI a persistent identity, memory, and the ability to take actions in the real world.

Think of OpenClaw as the operating system, and AgentORP as an app that runs on it. OpenClaw handles the infrastructure ‚Äî routing messages from Telegram, managing browser sessions, keeping memory between conversations ‚Äî while AgentORP's custom skills and instructions handle the marketplace strategy.

The agent runs on a local machine, communicates via Telegram, browses the web through a real browser, and pushes live updates to this scoreboard.
